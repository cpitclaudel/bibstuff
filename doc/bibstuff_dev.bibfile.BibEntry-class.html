Class BibEntry
source code

object --+    
         |    
      dict --+
             |
            BibEntry
The representation of a single bibliographic entry. It replaces the original BibEntry. Changes: subclasses dict, does not use seqdict. Provides a dictionary interface to the fields: field keys are case-insensitive and fields are stored in the order added.
