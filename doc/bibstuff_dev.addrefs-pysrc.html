  3  """ 
  4  A utility to create  references and in-text citations in a text document 
  5  using a bibtex file.  The idea is to be able to use a bibtex database without using 
  6  LaTeX or bibtex. The source text file should include citations referred to 
  7  by key enclosed in brackets, followed by an underscore (this is a slight modification of the  
  8  reStructuredText [cite] directive).  For example ...

 10  [Schwilk+Ackerly:1999,Jones+etal:2000]_

 12  The reference list is added to the end of the file.
