A default style for bib4txt.py Produces a list of citations that to be included in a reStructuredText document. (In very simple documents, can also substitute in the text for the citation references.)

A style includes:

  • citation template
  • citeref template
  • sortkey for make_text_output (often based on a field list)
  • name formatting template (e.g., 'v |l,| j,| f{. }.')


Notes:

  • you will often want to override these
  • shared.py holds defintions common to most styles
  • see the examples (e.g., example_numbered.py) of different styles
