Module addrefs
source code

A utility to create references and in-text citations in a text document using a bibtex file. The idea is to be able to use a bibtex database without using LaTeX or bibtex. The source text file should include citations referred to by key enclosed in brackets, followed by an underscore (this is a slight modification of the reStructuredText [cite] directive). For example ... [Schwilk+Ackerly:1999]_ separate multiple references by commas. For example . . . [Schwilk+Ackerly:1999,Jones+etal:2000]_

The reference list is added to the end of the file.
