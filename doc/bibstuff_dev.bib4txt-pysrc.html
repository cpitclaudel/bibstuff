  2  ''' 
  3  Creates formatted references for a text document (useful for reStructuredText documents). 
  4  Interacts with a Bibtex style database file 
  5  (without using LaTeX or bibtex). 
  6   
  7  The source text file should include citation references in reStructuredText format: 
  8  a citation key enclosed in brackets, followed by an underscore. 
  9  Citation keys cannot be all digits. 
 10  The source document can be output with formatted citation references substituted for the citation keys. 
 11  In this case, the reference list is added to the end of the file.    
 12   
 13  A slight modification of the reStructuredText ``cite`` directive is currently allowed: 
 14   
 15  - Most characters are permitted. 
 16    E.g., ``[Schwilk+Isaac:2006]_`` is not currently legal in reST but will be recognized. 
 17  - Comma separted multiple keys are permitted in a cite:  e.g., ``[Schwilk1999,Isaac2000]_`` 
 18   
 19  The intent is for the formatted references to be written to a separate file. 
 20  You can then include this in your reST document with an ``include`` directive. 
 21   
 22  How it works: 
 23   
 24  - Uses SimpleParse_ to convert an EBNF_ grammar into an object for scanning reST files for citation references. 
 25  - Uses SimpleParse_ to convert an EBNF_ grammar into an object for scanning .bib files.  (See Bibstuff's bibgrammar.py.) 
 26  - Extracts the citation references from the input document. 
 27  - Outputs a sorted list of citation definitions, to be used in the References section of your documents.
