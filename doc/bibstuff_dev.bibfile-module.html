Module bibfile
source code

Provides two classes, BibFile and BibEntry for accessing the parts of a bibtex database. BibFile inherits from simpleparse.dispatchprocessor. To fill a BibFile instance, i, call bibgrammar.Parse(src, i).

This framework could still be made more general along the lines of BtParse.
