Module Hierarchy

  • bibstuff: __init__.py for bibstuff package
    • bibstuff.addrefs: A utility to create references and in-text citations in a text document using a bibtex file.
    • bibstuff.bib4txt: Creates formatted references for a text document (useful for reStructuredText documents).
    • bibstuff.bibfile: Provides two classes, BibFile and BibEntry for accessing the parts of a bibtex database.
    • bibstuff.bibgrammar: bibgrammar module: provides an EBNF description of the bibtex bibliography format.
    • bibstuff.biblabel: Simple script to generate automatic labels (keys) for bibtex database entries.
    • bibstuff.bibname: Parses bibtex-formatted author/editor names lists and provides formatting functions (often via bibstyles/shared.NamesFormatter).
    • bibstuff.bibsearch: Utility for extracting reference(s) from bibtex database file.
    • bibstuff.bibstyle: This module is retained mostly to support addrefs.py, which is possibly superceded by makebib4rst.py
    • bibstuff.bibstyles: __init__.py for bibstyles package
      • bibstuff.bibstyles.default: A default style for bib4txt.py Produces a list of citations that to be included in a reStructuredText document.
      • bibstuff.bibstyles.default_templates: Provides default templates for style writers.
      • bibstuff.bibstyles.example_numbered: Provides an examle of how to easily modify an existing style: Just import everything from that style, and then override what you wish to change.
      • bibstuff.bibstyles.shared: Utilities and formatting classes for BibStuff, especially for bib4txt.py.
    • bibstuff.jabbrev: A utility to read journal abbreviations from a text file and modify a bibtex database to use the abbreviations.
    • bibstuff.reflist: Produces a list of reference keys from a .bbl file created by bibtex.
    • bibstuff.reformat_para: Provides paragraph reformatting.

Class Hierarchy

  • bibstuff.bibstyle.BibStyle: The base class.
  • object: The most base type
    • bibstuff.bibstyles.shared.NamesFormatter
    • dict: dict() -> new empty dictionary.
      • bibstuff.bibfile.BibEntry: The representation of a single bibliographic entry.
    • bibstuff.bibstyles.shared.CitationManager
      • bibstuff.bibstyles.example_numbered.CitationManager
      • bibstuff.bibstyles.default.CitationManager
    • bibstuff.bibstyles.shared.NameFormatter: Create a NameFormatter object based on a template string.
  • simpleparse.processor.MethodSource: Base class for MethodSource objects (including Processors and Parsers) Most applications will use either Processor or Parser objects, rather than directly using a MethodSource object.
    • simpleparse.processor.Processor: Provides definition of a generic processing API
      • simpleparse.dispatchprocessor.DispatchProcessor: Dispatch results-tree in a top-down recursive pattern with attribute lookup to determine production -> method correspondence.
        • bibstuff.addrefs.CitationFormatter: Formats citations
        • bibstuff.bibname.BibName: Parses a bibtex names entry (author, editor, etc).
        • bibstuff.bibstyles.shared.CiteRefProcessor: Formats inline citations and substitutes them into text.
          • bibstuff.bibstyles.example_numbered.CiteRefFormatter
          • bibstuff.bibstyles.default.CiteRefFormatter: Each style must inlcude a CiteRefFormatter class, which must define methods for each production (see the help for simpleparse.dispatchprocessor).
        • bibstuff.bibfile.BibFile: Stores parsed bibtex file.
